# caseflows

A reader's guide to international legal cases between Ukraine and Russia, 2014–present.

Static site — concept sketches for the database landing page. Deployed on Vercel.

## Structure

- `index.html` — concept index (live concepts list)
- `concepts/` — individual landing-page concept sketches
- `cases.json` — structured dataset (41 cases, 11 forums)
- `research/` — moodboard and landing-page study notes
